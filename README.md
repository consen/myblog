Myblog
======

##「不积跬步，无以至千里；不积小流，无以成江海」

My blog based on Hexo, see the generated files at [consen.github.io](https://github.com/consen/consen.github.io) repository.

Getting started
---------------

Install dependencies:

``` bash
$ git clone https://github.com/consen/myblog
$ cd myblog
$ npm install
```

Generate:

``` bash
$ hexo generate
```

Run server:

``` bash
$ hexo server
```

Deploy:

``` bash
$ hexo deploy
```
